Getting Started

Three steps to unified knowledge

Add your sources

Point Frozen Ink at your GitHub repos, Git repositories, Obsidian vaults, or MantisHub instances.

fink add github \ --name my-repo \ --owner acme \ --repo backend

Sync & serve

Frozen Ink crawls your sources, indexes everything with full-text search, and renders clean markdown.

fink sync my-repo \ && fink serve

Use everywhere

Browse in the web UI, query from AI assistants via MCP, or publish to Cloudflare for team access.

fink publish my-repo \ --name team-docs
