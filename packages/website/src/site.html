Why Frozen Ink

Five key capabilities for modern knowledge workflows.

From offline access to AI-ready context, Frozen Ink is built to move knowledge across boundaries without losing fidelity.

Online to Offline

Keep offline access to online systems like GitHub and MantisHub by syncing them into a local knowledge base.

Offline to Online

Make local knowledge, including Obsidian notes, available to cloud agents through MCP-powered deployments.

Optimized for AI

Frozen Ink is built for LLM workflows with markdown-native output, free-text search, and MCP access out of the box.

Break the Silos

Efficiently gather knowledge across many services into one unified workspace you can browse, search, and query.

Archival

Create durable snapshots of your knowledge for long-term archival, compliance, and historical reference.

How it works

Three steps to unified knowledge

Get up and running in minutes. No configuration files, no cloud accounts required.

Add your sources

Point Frozen Ink at your GitHub repos, Obsidian vaults, Git repositories, or MantisBT instances.

fink add obsidian --name my-vault --path ~/Vault

Sync & search

Frozen Ink crawls your sources, indexes everything with full-text search, and renders clean markdown.

fink sync "*" && fink serve

Use everywhere

Browse in the web UI, query from AI assistants via MCP, or publish to Cloudflare for team access.

fink publish my-vault --name team-docs
